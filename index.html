Introduction

HTML (HyperText Markup Language) is the fundamental language used to create web pages. It structures content using elements and tags, allowing browsers to interpret and display text, images, links, and more. This technical documentation will explore essential HTML concepts, including document structure, common elements, attributes, and forms, providing a foundation for building and understanding web pages.

This documentation introduces core HTML concepts such as elements, attributes, document structure, and tags.

Prerequisites

Before starting with HTML, you should have basic familiarity with using computers and browsing the web. It’s important to know how to set up a basic development environment using a text editor and web browser, as well as understanding how to create and manage files on your system.

While not required, it’s helpful to have prior experience working with web pages and basic file structure concepts. Much of the foundational information about HTML can be revisited as needed.

Once you've grasped HTML, you can explore more advanced web development topics like:

  • CSS for enhancing the appearance of your web pages, such as altering text size, choosing fonts, adding borders, creating multi-column layouts, and incorporating animations.
  • JavaScript for adding interactive and dynamic elements to your site, like showing or hiding content based on user input, updating elements in real-time, or storing user data locally.
Basics

HTML is the foundation of web development. It provides the basic structure for web pages using elements and tags that browsers interpret to display content. Understanding HTML is crucial for building web pages, as it allows you to organize and format text, embed multimedia, and create links.

In this section, you'll learn about essential HTML elements, including headings, paragraphs, links, images, and lists. You'll also discover how to use attributes to provide additional information about elements and how to structure your HTML documents for better readability and maintainability.

  • Elements: The building blocks of a web page, defining its structure and content. Each element includes a start tag, content, and an end tag. Together, these elements create the layout and functionality of a web page, and are interpreted by browsers to render the page visually and interactively.
  • Attributes: Provide essential details about elements, influencing their behavior and appearance. Specified as name-value pairs within an element's start tag, attributes like href in a link or src in an image define key properties such as destination URLs or file paths. By using attributes, developers can customize how elements are displayed, enhance accessibility, and control interactions, thereby creating more dynamic and functional web pages.
  • Document Structure: Arranges elements into a clear hierarchy, providing organization and flow for the content on a web page. It helps ensure that the page is properly understood by browsers, search engines, and assistive technologies, allowing for efficient navigation and accessibility.
  • Tags: define the start and end of an HTML element, instructing the browser on how to display content on a web page. Tags are enclosed in angle brackets, like <h1> and </h1>, and most elements have an opening tag and a closing tag. The content between the tags is the data or information that the browser processes and renders on the page. For example, <p> creates a paragraph, and <img> displays an image. Tags can also include attributes to modify the behavior or appearance of an element, like specifying the source of an image or adding links.
Elements

Elements are the building blocks of web pages. An element typically consists of an opening tag, content, and a closing tag. For example, a paragraph element is created using <p> and </p>, with the text appearing between the tags.

Elements define the structure and content of a web page, including text, images, links, and more. They are essential for organizing content in a readable and accessible way. Elements can also be nested within one another to create more complex structures.

  • Block Elements: These elements take up the full width of their container and start on a new line. Examples include <div>, <p>, and <h1>.
  • Inline Elements: These elements only take up as much space as necessary and do not start on a new line. Examples include <span>, <a>, and <img>.
  • Nesting Elements: HTML elements can be nested within other elements to create structured layouts. For example, a <div> may contain multiple <p> elements.
  • Self-Closing Elements: Some elements do not have closing tags, such as <br> for line breaks or <img> for images.
Attributes

HTML attributes provide additional information about an element. They appear inside the opening tag and usually come in name/value pairs like name="value".

Common attributes include:

  • class: Specifies a class for CSS styling.
  • id: Specifies a unique ID for an element.
  • src: Specifies the source of an image.
  • href: Specifies the URL of a link.

Example of an image element with the src attribute:

<img src="image.jpg" alt="An example image">
Document Structure

An HTML document typically starts with a <!DOCTYPE html> declaration to define the document type.

It is structured with a <html> element containing two main sections:

  • <head>: Metadata, such as the title and character encoding.
  • <body>: The visible content of the web page.

A simple HTML document looks like this:


          <!DOCTYPE html>
          <html>
            <head>
              <title>My Page</title>
            </head>
            <body>
              <h1>Hello, world!</h1>
            </body>
          </html>
Tags

HTML tags are the basic building blocks of HTML. Tags are used to define and structure the content on a web page. Each tag is enclosed in angle brackets, like <tagname>.

Some common HTML tags include:

  • <h1> to <h6>: Headings of different levels.
  • <p>: Paragraphs of text.
  • <a>: Links to other pages or resources.
  • <img>: Images.
  • <ul> and <ol>: Unordered and ordered lists.

HTML tags often come with attributes that provide additional information or modify their behavior. For example:

<a href="https://example.com">Visit Example</a>
